# Import

You can import any FWCloud exported previously. To import a FWCloud, press Import button


You will need the '*.fwcloud file' generated when it was exported. We can drag an drop and drop the file in the box:


, or choose the file picker:


Once the file has been added we proceed clicking the button Import FWCloud


While the importing is in process this form change so we can see progress


When the process is finish the form will close and a pop up messages indicates the result of the import
