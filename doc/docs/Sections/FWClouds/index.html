# New

A FWCloud is a logical grouping of firewalls, firewall clusters, IP objects, VPN connections, etc.

You can have a FWCloud for each of your clients or, for example one FWCloud for each country in a multinational company.

To create a new FWCloud, press + New button


and fill the form with the name of the new FWCloud


When the new fwcloud is created you can see a message tip telling how to create a new firewall or cluster for this FWCloud
